# Import Examples

This directory contains CSV files pre-populated with mock data to illustrate the correct structure, format, and relationship keys required by the import scripts.

> [!WARNING]
> **Mock Data Reference Only**
> - The rows in these files contain strictly **fake data** (names, phone numbers, and emails).
> - Do not replace the contents of these files with real school records.
> - Do not commit real student or staff data to the repository.
